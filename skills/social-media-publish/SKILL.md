# Social Media Publish Skill

Use this skill when the task is to schedule posts, publish approved content, handle comments and DMs, or report platform metrics for the content matrix.

## When to use

- Task asks for "排期发布", "发布", "评论互动", or "数据回收".
- Content calendar or CMO assigns publish dates and approved drafts.
- Need to reply to comments/DMs with consistent tone and escalate when needed.

## Steps

1. **From Paperclip**: Read the task (publish list, platforms, dates) and any approved drafts or links. Check company/content guidelines in task or wiki.
2. **Schedule**: Update the schedule (e.g. spreadsheet or shared doc). For each slot: platform, time, asset, caption. If the task is "generate publish list", output the list in a task comment or attached doc.
3. **Publish**: If the task includes "执行发布" and you have access to publishing tools/APIs, publish per the schedule. Otherwise, produce a clear publish checklist for a human or tool to execute.
4. **Engage**: For comment/DM reply tasks: use approved templates and tone; do not promise features or data that are not confirmed. Escalate sensitive or unclear cases to CMO in task comments.
5. **Metrics**: Collect platform metrics (views, likes, saves, conversions if available). Summarize in task comment or agreed format; add short attribution notes and feed back to CMO.

## Collaboration

- Do not change copy or creative without approval; publish only approved content.
- Sensitive or legal issues: escalate in task comments and do not reply publicly until CMO or board confirms.
