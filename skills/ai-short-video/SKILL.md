# AI Short Video Skill

Use this skill when the task is to create short-video scripts, voiceover outlines, or shot lists for platforms like Douyin (抖音) or WeChat Channels (视频号).

## When to use

- Task asks for "短视频脚本", "口播提纲", or "分镜".
- Content calendar or CMO assigns a short-video slot with topic and duration.
- Need hook, structure, and CTA for a short-form video.

## Steps

1. **From Paperclip**: Read the task and any brief (topic, duration, platform, tone). Check linked issues or comments for product messaging and CTA.
2. **Structure**: Draft a script with:
   - **Hook (first 3 seconds)**: One line or visual that grabs attention.
   - **Body**: Shots/scenes with voiceover or caption bullets; keep pace and platform norms in mind.
   - **CTA**: Clear call-to-action at the end (e.g. download, follow, comment).
3. **Deliverables**: Provide in task comment or attachment:
   - Script (voiceover text + optional shot list).
   - Caption highlights and BGM/rhythm notes if relevant.
4. **Handoff**: Mark script as ready for review; production or ops will handle filming/editing or integration with video-generation APIs.

## Collaboration

- Do not film or publish; output is script and outline only unless the task explicitly includes production.
- Align messaging with product and brand; do not invent features or data. If in doubt, ask CMO or content lead in task comments.
